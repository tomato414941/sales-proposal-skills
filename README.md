# sales-proposal-skills

営業担当者の少ない入力から提案に必要な情報を整理し、不足情報を確認しながら、最終的に組み込みの PPTX 機能で顧客向け提案資料を作る skill です。

## 前提条件

- Claude の skill 実行環境
- 組み込み `pptx` スキルが利用できること

## 使い方

案件情報、議事録、RFP、製品情報などを渡して、提案資料を作りたいと伝えてください。

この skill は次の順で進みます。

1. 入力から提案の骨格を整理
2. スライド作成前に、不足情報をまとめて確認
3. 提案戦略とスライド構成を設計
4. 内容を批判的に見直し
5. 最後に PPTX を生成
6. 生成後に持参品質を確認

主な成果物:

- 追加確認が必要な場合は、その確認事項
- 提案戦略
- スライド構成
- 最終 PPTX

Claude Code では、必要に応じて `.proposal/` に途中成果物を残せますが、必須ではありません。
